3.11 付録: 多様体

多様体 \(M\) とは、各開近傍から \(R^n\) の開集合への連続した 1-1 写像を持つ点の集合です。簡単に言えば、多様体 \(M\) は 局所的に 標準的な \(R^n\) に似ています。M の各開近傍から \(R^n\) へのこの写像は、\(M\) の各点 \(P\) に n 組の \((x_1(P), ...x_n(P))\) を関連付けます。ここで、数 \(x_1(P), ...x_n(P)\) は点 \(P\) の座標と呼ばれます。したがって、n 次元多様体は、任意の点のある近傍に n 個の独立座標を与えることができる集合であると考えることもできます。

多様体の例として、球面が挙げられます。三次元球面は二次元球面 \(S^2\) と呼ばれ、\(R^3\) 内の点の集合において \(x^2 + y^2 + z^2 = r\) が成り立つものとして定義されます。ここで、\(r\) は球面の半径です。三次元球面は二次元であることに注意してください。これは、定義に3つの座標と1つの条件が含まれており、これにより自由度が1つ除去されるためです。そのため、数学的には二次元球面と呼ばれます。球面が多様体であることを確認するには、\(R^2\) への写像が必要です。この写像は通常の球座標によって与えられます。

図3.9: 球面のある近傍から\(R^n\)への写像の図解

球面上のほぼすべての点は、\((ϕ, θ)\) という形式の座標の組み合わせで明確に識別できます。ほぼすべてです！ 極 \(ϕ = 0\) はどこに写像されるのでしょうか？ 図に示すように、極は直線全体に写像されるため、1対1の識別は不可能です。したがって、この写像は完全な球面には適用できず、極の近傍にあるものを記述するには、別の写像が必要です。同様の問題が半円 \(θ = 0\) 上の写像でも発生します。各点は \(R^2\) 内で \(θ = 0\) と \(θ = 2π\) に写像できますが、これも1対1の写像ではありません。これは、多様体では一般に、多様体のすべての点に対して1つの座標系が存在するのではなく、ある近傍で有効な局所座標のみが存在することを示しています。これは問題ではありません。 多様体の定義的特徴は、局所的には \(R^n\) に似ているということだからです。

球面座標写像は開近傍\(0 < ϕ < π, 0 < θ < 2π\) においてのみ有効であり、球面全体をカバーするには2つ目の写像が必要です。例えば、異なる向きを持つ2つ目の球面座標系を用いることで、問題となる極はこの写像では異なる点に位置し、\(ϕ = 0\) には位置しなくなります。この2つ目の写像を用いることで、球面上のすべての点は \(R^2\) への写像を持ち、2次元球面は多様体として見ることができます。

多様体の簡単な例はもちろん\(R^n\)である。
