3.7.1 ローレンツ群の一つの表現

ローレンツ群の定義条件(式3.133)を用いて、許容される変換の明示的な行列表現を構築する方法を見てみましょう。まず、何を求めようとしているのか少し考えてみましょう。ローレンツ群は、4元ベクトル123に作用する場合、実4×4行列で与えられます。これらの行列は実数でなければなりません。なぜなら、実ミンコフスキー空間\(\mathcal{R}^{(1,3)}\)の要素にどのように作用するかを知りたいからです。一般的な実4×4行列には16個のパラメータがあります。ローレンツ群の定義条件は実際には10個の条件124であり、これを6個のパラメータに制限します。言い換えれば、最も一般的なローレンツ変換を記述するには、6個のパラメータが必要です。したがって、6個の線型独立な生成子を見つければ、この群のリー代数の完全な基底が見つかったことになります。これは、他のすべての生成子がこれらの基底生成子の線型結合として書けることを意味します。さらに、これらの基底生成子をリー括弧に入れたときにどのように振る舞うかを計算し、このリー代数の抽象的な定義を導くことができます。

3.7.4 \((0, 0)\) 表現

最も低次元の表現は、\(\mathcal{SU}(2)\)の場合と同様に自明である。 \(\mathcal{su}(2)\)の両方のコピーにおいてベクトル空間は1次元だからである。 したがって、生成子は1×1行列でなければならず、交換関係を満たす唯一の1×1「行列」は数0である。 \[ N_i^+ = N_i^− = 0 → e^{iN_i^+} = e^{iN_i^−} = e^0 =1 \tag{3.176} \] したがって、ローレンツ群の \((0, 0)\) 表現は、ローレンツ変換によって変化しないオブジェクトに作用すると結論付けられます。この表現はスカラー表現と呼ばれます。

この表現は、\((\frac{1}{2}, 0)\)表現と同様に構築できますが、今回は\(N_i^+\)の1次元表現、つまり\(N_i^+ = 0\)を使用し、\(N_i^−\)の2次元表現、つまり\(N_i^− = \frac{1}{2}σ_i\)を使用します。この表現は\((\frac{1}{2}, 0)\)表現と全く同じように見えると最初は思われますが、そうではありません！今回は、式3.169のN+の定義から、 \[ \begin{align} &N_i^+ = \frac{1}{2}(J_i + iK_i) =0 \tag{3.188} \\ \\ &→ J_i = −iK_i \tag{3.189} \end{align} \] マイナス記号に注意してください。3.6.3節で導出した\(N^+\)の\(\mathcal{su}(2)\)の2次元表現を用いると、次のようになります。 \[ N_i^− =\frac{1}{2}σ_i =\frac{1}{2}(J_i − iK_i)\underbrace{=}_{Eq. 3.189}\frac{1}{2}(−iK_i − iK_i) = −iK_i \tag{3.190} \] これを用いてブーストジェネレータの\((0,\frac{1}{2})\)表現を推定することができる。 \[ −iK_i = \frac{1}{2}σ_i → Ki = \frac{−1}{2i}σ_i =\frac{−i}{2i^2}σ_i = \frac{i}{2}σ_i \tag{3.191} \] さらに、式3.189から次の式が得られる。 \[ J_i = −iK_i = \frac{1}{2}σ_i \tag{3.192} \] この表現ではローレンツ回転は次のように表される。 \[ R_θ = e^{i\vec{θ}\cdot\vec{J}} = e^{i\vec{θ}\cdot\frac{\vec{σ}}{2}} \tag{3.193} \] そしてローレンツブースト \[ B_\phi = e^{i\vec{\phi}\cdot\vec{K}} = e^{−\vec{\phi}\cdot\frac{\vec{σ}}{2}} \tag{3.194} \] したがって、回転は \((\frac{1}{2} , 0)\) 表現と同じですが、ブーストは指数のマイナス符号が異なります。両方の表現は、類似しているが同じではないオブジェクトに作用すると結論付けられます。これらのオブジェクトを、右カイラルスピノルに対するローレンツ群作用の \((0, \frac{1}{2})\) 表現と呼びます。 \[ χ_R = \begin{pmatrix} (χ_R)^1 \\ (χ_R)^2 \end{pmatrix} \tag{3.195} \] 左カイラル スピノルと右カイラル スピノルの総称は ワイル スピノル です。

3.7.7 ファン・デル・ヴェルデン記法

ここで、スピノルを扱うのに非常に便利な表記法を導入します。2種類のオブジェクトは変換の仕方が異なるため、区別する必要があることが分かっています。前のセクションでは、それらは異なるものの、それほど大きくは変わらないことを学びました。すぐに、\((\frac{1}{2}, 0)\) 表現に従って変換するオブジェクト (左カイラルスピノル) と、\((0, \frac{1}{2})\) 表現に従って変換するオブジェクト (右カイラルスピノル) の間に関連があることを学びます。これらの異なるオブジェクトを1つの表記法で記述できるように、ドット付きインデックスとドットなしインデックスの概念を導入します。これは、発明者にちなんで、ファン・デル・ヴェルデン表記法と呼ばれることもあります。これにより、どのオブジェクトがどのように変換されるかを追跡できるようになります。これは、完全な形式論を構築すれば、すぐにはるかに明確になります。

左カイラルスピノル\(χ_L\)は下側の点なしの指数を持つと定義しよう。 \[ χ_L = χ_a \tag{3.196} \] 右カイラルスピノル\(χ_R\)は上付き点線インデックスを持つ。 \[ χ_R = χ^{\dot{a}} \tag{3.197} \] 次に、「スピノル計​​量」を導入します。スピノル計​​量により、右カイラルスピノルを左カイラルスピノルに変換したり、その逆を行ったりすることが可能になりますが、後ほど説明するように、それだけでは十分ではありません。スピノル計​​量150を以下のように定義します。

3.7.8 \((\frac{1}{2}, \frac{1}{2})\) 表現

この表現では、\(\mathcal{SU}(2)\) リー代数158 \(N_i^+\) と \(N_i^-\) の両方のコピーに対して2次元表現を使用します。今回は、まずこの表現がどのようなオブジェクトに作用するのかを見てみましょう。\(N_i^+\) と \(N_i^-\) は可換、つまり \([N_i^+, N_j^-] = 0\) (式 3.172) であるため、コピーは互いに干渉しません。したがって、オブジェクトは両方のコピーの下で別々に変換されます。調べたいオブジェクトを \(v\) と名付けましょう。このオブジェクトには2つのインデックス \(v_a^{\dot{b}}\) があり、それぞれが \(\mathcal{su}(2)\) の別々の2次元コピーの下で変換されます。ここで、前のセクションで紹介した表記法が役立ちます。

158 数学的には、 \((\frac{1}{2}, \frac{1}{2}) = (\frac{1}{2}, 0) ⊗ (0, \frac{1}{2})\) となります。

3.7.9 スピノルとパリティ

ここまでのところ、\((\frac{1}{2}, 0)\) 表現に従って変換するオブジェクトを左カイラル、\((0, \frac{1}{2})\) 表現に従って変換するオブジェクトを右カイラルと呼ぶ理由が分かりません。パリティ変換について少し説明すれば、この意味が理解できるでしょう。

パリティ変換下におけるローレンツ群の生成子の振る舞いは既に分かっていることを思い出してください。その結果は 式3.161で、便宜上ここで引用します。 \[ J_i \underbrace{→}_{P}J_i　K_i \underbrace{→}_{P}−K_i \tag{3.230} \] 式3.169の生成子\(N^±\)の定義を見ると、 ここでも引用されている \[ N_i^± = \frac{1}{2}(J_i ± iK_i) \tag{3.231} \] パリティ変換の下では \(N^+ ↔ N^-\) となることがわかります。したがって、ある変換の \((0, \frac{1}{2})\) 表現は、この変換の \((\frac{1}{2} , 0)\) 表現になり、パリティ変換の下ではその逆も成り立ちます。これが、左手系と右手のカイラルスピノルについて話す理由です172。パリティ変換の下で右手系の座標系が左手系の座標系に変化するのと同様に、これら2つの表現は互いに変化します。

172 慣用的な名称は左巻きスピノルと右巻きスピノルですが、これは非常に混乱を招く可能性があります。なぜなら、左巻きと右巻きという概念は、カイラリティとは異なるヘリシティという概念に直接関連しているからです。いずれにせよ、パリティ変換によって左巻きのものが右巻きのものに変わるので、この名称には意味があるはずです。

回転はどちらの表現でも同じですが、ブースト変換は符号が異なるため、上記のステートメントを明示的に記述するのは簡単です。 \[ \begin{align} (Λ_{\vec{K}})_{(\frac{1}{2},0)} &= e^{\vec{\phi}\vec{K}} \underbrace{→}_{P} e^{-\vec{\phi}\vec{K}} = (Λ_{\vec{K}})_{(0,\frac{1}{2})} \tag{3.232} \\ \\ (Λ_{\vec{K}})_{(0,\frac{1}{2})} &= e^{-\vec{\phi}\vec{K}} \underbrace {→}_{P} e^{\vec{\phi}\vec{K}} = (Λ_{\vec{K}})_{(\frac{1}{2},0)} \tag{3.233} \end{align} \] ここで、パリティ変換に対して不変な物理系を記述したい場合、常に右カイラルスピノルと左カイラルスピノルが必要になることを学びます。最も簡単な方法は、それらをディラックスピノルと呼ばれる単一のオブジェクトに重ねて記述することです。 \[ Ψ = \begin{pmatrix} χ_L \\ ξ_R \end{pmatrix} = \begin{pmatrix} χ_a ξ^{\dot{a}} \end{pmatrix} \tag{3.234} \] 左カイラルスピノルと右カイラルスピノルの総称がワイルスピノルであることを思い出すと、ディラックスピノル \(Ψ\) は2つのワイルスピノル \(χ_L\) と \(ξ_R\) から構成されると言えます。ここでは一般論に留まり、\(χ\) と \(ξ\) の間に先験的な関係を仮定しないことに注意してください。次の形式のディラックスピノルは、 \[ Ψ_M = \begin{pmatrix} χ_L \\ χ_R \end{pmatrix} \tag{3.235} \] これは特別な場合であり、マヨラナスピノルと呼ばれます。ディラックは4元ベクトルではありません。なぜなら、全く異なる変換を行うからです173。ディラックスピノルはローレンツ群の \((\frac{1}{2}, 0) ⊕ (0, \frac{1}{2})\) 表現174に従って変換されます。これは、対応する変換をブロック対角形式で1つの大きな行列に書き込むことを意味します。

3.7.10 スピノルと電荷共役

3.7.7節では、\(χ_L → χ_R\) と \(ξ_R → ξ_L\) という変換に遭遇しました。この変換は \(χ_L → χ_L^{\mathcal{C}} = \epsilonχ_L^* = χ_R\) によって実現され、右カイラルスピノルについては同様に \(ξ_R → ξ_R^{\mathcal{C}} =(-\epsilon))ξ_R^* = ξ_L\) となります。これで、全く異なる視点から理解できるようになりました。
