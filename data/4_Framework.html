4 フレームワーク(枠組み)

この章の基本的な考え方は、何かを最小化することで、正しい自然方程式を得るというものです。この何かとは何でしょうか？ 一つ確かなことは、対象はローレンツ変換によって変化してはならないということです。そうでなければ、異なる座標系に対して異なる自然法則が導かれるからです。数学的に言えば、これは、私たちが探している対象はスカラー、つまりローレンツ群の(0, 0)表現に従って変換する対象でなければならないことを意味します。可能な限り単純な選択という制約と合わせて、これで正しい自然方程式を導くのに十分でしょう。自然は単純なものを好むのです。

この考え方に基づいて、ラグランジュ形式と呼ばれる枠組みを紹介します。理論の中心となる対象を最小化することで、対象の物理系を記述する運動方程式が得られます。この最小化手順の結果は、オイラー・ラグランジュ方程式と呼ばれます。

ラグランジアン形式論は、現代物理学における最も重要な定理の一つであるネーターの定理を導くことを可能にします。この定理は、対称性と保存量1の間に深いつながりがあることを明らかにしています。次章ではこのつながりを用いて、実験で測定する量が理論によってどのように記述できるかを理解します。

1 保存量とは、時間的に変化しない量のことです。よく知られている例としては、ある系のエネルギーや運動量などが挙げられます。数学的には、これは \(∂_tQ = 0 → Q =一定\) を意味します。

4.1 ラグランジアン形式

ラグランジアン形式論は、基礎物理学のほとんどの分野で用いられる非常に強力な枠組み2です。この形式論は比較的単純です。なぜなら、基本対象であるラグランジアンはスカラー3だからです。この形式論は、対称性を考慮したい場合に非常に有用です。ラグランジアン上の積分である作用が、ある対称性変換に対して不変であることを要求すると、問題のシステムのダイナミクスがこの対称性を尊重することが保証されます。

2 もちろん、他にも枠組みは存在します。例えば、ハミルトニアン形式論は、ハミルトニアンを中心対象としています。ハミルトニアンの問題は、それが表すエネルギーが共変エネルギー・インパルス・ベクトルの単なる一要素に過ぎないため、ローレンツ不変ではないことです。

3 スカラーとは、ローレンツ群の(0, 0)表現に従って変換されるオブジェクトです。つまり、ローレンツ変換によって全く変化しないということです。

4.1.2 変分法 - 基本的な考え方

関数の極小値を求めることができる新しい理論を開発したいのであれば、一歩下がって数学的な極小値を特徴付けるものは何かを考える必要があります。変分法の答えは、極小値はその近傍によって特徴付けられるということです。例えば、普通の関数\(f(x) = 3x^2 + x\)の極小値 \(x_{min}\) を求めてみましょう。まず、特定の\(x = a\)に注目し、その近傍を詳しく見ていきます。数学的には、これは\(a + \epsilon\)を意味します。ここで、\(\epsilon\)は無限小（正または負）の変化を表します。このaの周りの変化を関数\(f(x)\)に代入します。 \[ f(a +\epsilon) = 3(a +\epsilon)^2 + (a +\epsilon) = 3(a^2 + 2a\epsilon + \epsilon^2) + a +\epsilon \] ここで重要な点は、\(a\) が最小値となる場合、\(\epsilon\) の一次変動は必ず消滅するということです。そうでない場合、\(\epsilon\) を負の \(\epsilon < 0\) とすると、\(f(a +\epsilon)\) は \(f (a)\) より小さくなります。したがって、\(\epsilon\) に関するすべての線形項を集め、それらが消滅することを要求します。 \[ 3 · 2a\epsilon+\epsilon \overset{!}{=}0 → 6a + 1 \overset{!}{=}0 \] この方程式は次のように解けます。 \[ x_{min} = a = \frac{−1}{6} \] もちろん、これは導関数 \(f(x) = 3x^2 + x → f^\prime(x) = 6x + 1\) をゼロにした場合に得られる結果と全く同じです。通常の関数に関して言えば、これは同じことを行う別の方法にすぎません。しかし、変分計算はさらに関数の極値を求めることができます。これが一般的な作用関数でどのように機能するかは、後ほど説明します。

ラグランジアン形式の考え方は、光に対するフェルマー原理のような原理が質量を持つ物体にも存在するというものです。残念ながら、質量を持つ物体はフェルマー原理に単純に従いません。しかし、より一般的な仮説を試みることはできます。 \[ S[q(t)] = \int \mathcal{L} dt \] ここで、\(\mathcal{L}\) は一般にラグランジアンと呼ばれる非定数関数です。 この関数は光に対しては定数ですが、一般的なラグランジアンは、対象となる物体の位置 \(q(t)\) の関数です。 さらに、ラグランジアンは物体の速度に依存する場合があります: \(\mathcal{L} = \mathcal{L}(q(t), \frac{∂}{∂t} q(t))\)。これについては次のセクション7で詳しく説明します。このような関数に対する変分計算の考え方の用法を詳しく見る前に、2つの小さな点について説明しておく必要があります。

7 私たちの課題は、与えられたラグランジアンと初期条件に対して、可能な限り最小の作用を持つ経路 \(q(t)\) を見つけることです。 それを行う前に、問題の物理系を記述する正しいラグランジアンを見つける必要があります。ここで、前章で説明した対称性が役に立ちます。ラグランジアンがローレンツ群のすべての変換に対して不変であることを条件にすれば、正しいラグランジアンにたどり着くことができます。

4.2 制限事項

第1章1節で既に述べたように、現在の理論には制約があり、これまでのところ第一原理から動機づけることはできません。私たちが知っているのは、理にかなった理論を得るためには、第一原理を尊重しなければならないということだけです。

重要な制約の一つは、ラグランジアンでは最低の、非自明な微分しか使えないということです。ここで自明とは、システムのダイナミクス、つまり運動方程式に影響を与えないことを意味します。理論によっては1次微分、2次微分になるものもあります。ある理論の最低次数は、ラグランジアンがローレンツ不変8であるという条件によって決まります。そうでなければ、異なる参照系に対して異なる運動方程式が得られるからです。理論によっては、1次微分では不変項が得られないため、2次微分が最低次となります。

4.3 粒子理論と場の理論

現在、自然を記述する枠組みは2つあります。一つは、物理系を時間に依存する粒子の位置、すなわち\(\vec{q} = \vec{q}(t)\)で記述する粒子理論です。自然を直交座標で記述する必要がないため、\(x\)ではなく\(q\)という記号を使用します13。このような理論では、ラグランジアンは位置\(\vec{q}\)、速度\(∂_t\vec{q}\)、および時間\(t\)に依存します。

4.4 オイラー・ラグランジュ方程式

まず粒子理論から始めます。これは、粒子が2つの固定点の間をどのように移動するかを解明することを意味します。私たちが解かなければならない数学的問題は、作用が \[ S =\int_{t_1}^{t_2}\mathcal{L}\left(q(t),\frac{dq(t)}{dt}, t\right)dt \] は極値（最大値または最小値）です。 \[ \dot{q}(t) ≡ \frac{dq(t)}{dt} \] 上の例と同様に、\(q(t) = a(t)\)を固定し、この粒子の軌道を少し変化させます。 \[ a(t) + \epsilon(t) \] ここで\(\epsilon\)は再び無限小です。粒子を考えると、経路を変えることができるだけでなく、同様に速度\(\dot{a}(t) +\dot{\epsilon}(t)\)も\(\dot{\epsilon}(t) = \frac{d\epsilon(t)}{dt}\)で変化させることができます。

境界では、変換されたパスは変換されていないパスと等しくなければなりません。 \[ 0 = \epsilon(t_1) =\epsilon(t_2) \tag{4.5} \] なぜなら、私たちは作用積分を極値化する2つの固定点の間の経路を探しているからです。

4.5 ネーターの定理

ネーターの定理は、ラグランジアンの対称性はそれぞれ、ある保存量に直接関係していることを示しています。あるいは、少し異なる表現で言えば、物理学者が自然を記述するために一般的に用いる概念（保存量）は、対称性と直接結びついています。これは、科学史上最も美しい洞察の一つです。

4.5.2 場の理論におけるネーターの定理 - 時空対称性

場については、時空変換によって起こり得る異なる種類の変化を区別する必要があります。観測者 \(S^\prime\) は場 \(Ψ^\prime(x^\prime)\) を見ますが、観測者 \(S\) は場 \(Ψ(x)\) を見ます。これは同じ場ですが、単に別の視点から見たものであり、2 人の観測者は同じ数値的な場の成分を見ていません。この 2 つの異なる記述は、ローレンツ群の適切な変換によって関連付けられています。セクション 3.7.11 で場の表現を導入しました。これをここで使用します。(無限次元) 微分演算子表現は \(x → x^\prime\) を変化させます。つまり、この表現を用いることで、時空の異なる点や回転したフレームにおける場の成分を計算できるということです。ローレンツ群の有限次元表現は \(Ψ → Ψ^\prime\) に変化し、つまり場の成分を混合します33。

4.5.3 ローテーションとブースト

次に、回転とブーストに対する不変性について見ていきます。まず式4.30の2番目の部分から始め、その後、1番目の部分の含意について見ていきます。1番目の部分から1つの量、2番目の部分から1つの量が得られ、これらは一緒に保存されます。スカラー場には混ざり合う可能性のある成分がないため、スカラー場の場合、1番目の部分はゼロです。言い換えれば、スカラーの場合、3.7.4節で導出したローレンツ群の生成子の1次元表現 \(S_{μν} = 0\) を使用します。したがって、この節で導出するのは、スカラー場の完全な保存量です。

変換の2番目の部分は次のように与えられる。 \[ x_μ → x_μ^\prime = x_μ + δx_μ = x_μ + M_μ^σx_σ \tag{4.42} \] ここで、\(M_μ^σ\) は回転とブーストの生成子であり、その無限次元表現である。つまり、これらは式 3.248 で定義されている微分作用素によって表される。

式4.42から、座標の変化は\(δx_μ = M_μ^σx_σ\)で与えられることがわかります。これを式4.33に代入すると、 \[ ∂_ν\left(\frac{∂\mathscr{L}}{∂(∂_νΦ)}\frac{∂(Φ)}{∂x_μ}− δ_μ^ν\mathscr{L}\right)M^{μσ}x_σ =0 \tag{4.43} \] \[ \underbrace{→}_{Eq. 4.35}∂_νT_μ^νM^{μσ}x_σ = 0 \] これを少し書き直して、従来の形式に置き換えることもできます。 \[ \begin{align} &=\frac{1}{2}(∂^νT_ν^μ M_{μσ}x^σ + ∂^νT_ν^μ\underbrace{ M_{μσ}}_{=−M_{σμ}} x^σ) =\frac{1}{2}(∂^νT_ν^μ M_{μσ}x^σ − ∂^νT_ν^μ M_{σμ}x^σ) \\ \\ &\underbrace{=}_{\text{ダミーインデックスをリネーム}}\frac{1}{2}(∂^νT_ν^μ M_{μσ}x^σ − ∂^νT_ν^σ M_{μσ}x^μ) =\frac{1}{2}(∂^νT_ν^μ x^σ − ∂^νT_ν^σ x^μ)M_{μσ} \\ \\ &=\frac{1}{2} ∂_ν(T^{μν}x^σ − T^{σν}x^μ)M_{μσ} = 0 \\ \\ &→ with (J^ν)^{σμ} ≡ T^{μν}x^σ − T^{σν}x^μ → ∂_ν(J^ν)^{σμ} = 0 \tag{4.44} \end{align} \] 最後のステップで導入した量 \(J^ν\) はネーターカレントと呼ばれます。式 3.173 の定義から導かれる \(M_{μσ} = −M_{σμ}\) の反対称性により、\(M_{μμ} = 0\) となることに注意してください。 したがって、ゼロでない \(M_{μσ}\) ごとに 1 つずつ、6 つの異なる連続方程式 \(∂_ν(J^ν) = 0\) が見つかりました。式 4.39 と同じ議論から、時間的に保存される量は次の式であると結論付けられます。 \[ \mathcal{Q}^{μσ} =\int d^3x(T^{μ0}x^σ − T^{σ0}x^μ) \tag{4.45} \] 回転不変性40については、保存量は

4.5.4 スピン

次に、式4.30の最初の部分の含意について見ていきます。これは、回転から得られる保存量の導出で見落としていた部分です。ここで、 \[ δΦ =\epsilon_{μν}S^{μν}Φ(x) \tag{4.48} \] ここで、\(S^{μν}\) は、問題43における変換の適切な有限次元表現である。これにより、式4.29において、

43 有限次元表現は、場の成分の混合を担います。例えば、回転生成子の2次元表現\(J_i = \frac{1}{2}σ_i\)は、ワイルスピノルの成分を混合します。

\[ ∂_ρ\left(\frac{∂\mathscr{L}}{∂(∂_ρΦ)}\epsilon_{μν}S^{μν}Φ(x)\right) \tag{4.49} \] これにより、式4.46に新たな項が加わります。完全な保存量は、 \[ L^i =\frac{1}{2}\epsilon^{ijk}Q^{jk} =\frac{1}{2}\epsilon^{ijk} \int d^3x\left(\frac{∂\mathscr{L}}{∂(∂_0Φ)}S^{jk}Φ(x) + (T^{j0}x^k − T^{k0}x^j)\right) \tag{4.50} \] そして我々はこう書く \[ L^i = L_{spin}^i + L_{orbit}^i \tag{4.51} \]

最初の部分は新しいものですが、これまで検討してきた通常の軌道角運動量と類似している必要があります。なぜなら、この2つの項は加算され、同じ不変性を考慮する際に現れるからです。標準的な見方では、この保存量の最初の部分は、ある種の内部角運動量44であると考えられています。

44 後ほど、場が粒子を生成したり破壊したりする様子を見ていきます。スピン \(\frac{1}{2}\) 場はスピン \(\frac{1}{2}\) 粒子を生成します。これは素粒子の不変の性質です。そのため、「内部」という言葉が使われています。軌道角運動量は、2つ以上の粒子が互いの周りをどのように回転するかを表す量です。

4.5.5 場の理論におけるネーターの定理 - 内部対称性

この付録では、ブースト不変性から、例として挙げたラグランジアン \(\mathcal{L} = \frac{1}{2}m\dot{\vec{q}}^2\) に従う保存量を求めます。ブースト変換は \(q → q^\prime = q + δq = q + vt\) であり、\(v\) は一定速度を表します。このラグランジアンはブーストに対して不変ではありませんが、上で述べたように、これは全く問題ではありません。 運動方程式が変換に対して不変であるためには、ラグランジアンは任意の関数の全微分まで変化することが許されます。この条件は、ブースト変換 \(q → q^\prime = q + vt \underbrace{⇒}_{v=一定!}\dot{q}^\prime = \dot{q} + v\) では明らかに満たされており、したがって、ラグランジアン \[ \mathcal{L} = \frac{1}{2}m\dot{q}^2 \] は、以下によって変更されます \[ \begin{align} δ\mathcal{L} &= \mathcal{L}(q, \dot{q}, t) −\mathcal{L}(q^\prime, \dot{q}^\prime, t) = \frac{1}{2}m\dot{q}^2 − \frac{1}{2}mz{\dot{q}^\prime}^2 \\ \\ &=\frac{1}{2}m\dot{q}^2 − \frac{1}{2}m(\dot{q} + v)^2 = −m\dot{q}v − \frac{1}{2}mv^2 \end{align} \] これは関数の全微分であす53。

53 \(\frac{dG}{dt} = \frac{d}{dt} (−mqv − \frac{1}{2}mv^2t) = −m\dot{q}v −\frac{1}{2}mv^2\)、\(v\)と\(m\)は定数だからです。

\[ −G ≡ −mqv − \frac{1}{2}mv^2t \] 運動方程式は変化せず、保存量54を見つけることができると結論付けます。

54 これを確認する別の方法は、運動方程式を直接見ることです： \(m\ddot{q} = 0\)。変換は \(q → q^\prime = q + vt\) となり、\(v =const\) したがって、\(\dot{q} → \dot{q}^\prime = \dot{q} + v\) かつ \(\ddot{q} → \ddot{q}^\prime = \ddot{q}\) となります。

ブーストの場合、保存量は（式4.16） \[ J_{boost} =\underbrace{\frac{∂\mathcal{L}}{∂\dot{q}}}_{≡p} \underbrace{δq}_{=vt}−G = pvt − mvq − \frac{1}{2}mv^2t \] ここで、式4.16では右辺に零点があるので、各項からvを1つずつ消去することができます。これにより、 \[ \tilde{J}_{boost} = \underbrace{pt − \frac{1}{2}mvt}_{≡\tilde{p}t} −mq =\tilde{p}t − mq \tag{4.62} \]

これはかなり奇妙な保存量です。なぜなら、その値は開始時刻に依存するからです。適切なゼロ時刻を選ぶことで、この量はゼロになります。これは保存されるので、常にゼロのままです。

4.7 付録: 場の理論におけるブースト不変性から得られる保存量

場の理論におけるブースト不変性については、保存量として次の式があります。 \[ \mathcal{Q}^{0i} =\int d^3x\left(T^{00}x^i − T^{i0}x^0\right) \tag{4.63} \] これは式4.47で\(K_i = M_{0i}\)を使って導出したものです。

この量をどのように解釈すればよいでしょうか？ \(\mathcal{Q}^{0i}\) は保存されます。つまり、 \[ 0 =\frac{∂\mathcal{Q}^{0i}}{∂t}=\underbrace{\int d^3x \frac{∂T^{i0}}{∂t}x^0}_{=x^0\frac{∂}{∂t}\int d^3xT^{i0}} +\int d^3xT^{i0}\underbrace{\frac{∂x_0}{∂t}}_{=1\; \because\; x_0=t} −\frac{∂}{∂t}\int d^3x \frac{∂T^{00}}{∂t}x^i \] \[ = t\frac{∂P^i}{∂t}+ P^i −\frac{∂}{∂t}\int d^3xT^{00}x^i \] 式4.20から、\(P^i\)は保存される\(\frac{∂P^i}{∂t} = 0 → P^i = 一定\)であることが分かります。 そして結論として、 \[ \frac{∂}{∂t} \int d^3xT^{00}x^i = P^i = 一定 \tag{4.64} \] したがって、この保存則は55エネルギーの中心が等速度で移動することを示しています。あるいは別の見方をすれば、運動量は全エネルギーと質量中心のエネルギーの速度の積に等しいと言えます。

55 この用語の意味についてはあまり心配しないでください。これは、これまで導出した他の保存量のように、実際には有用な保存量ではないからです。

しかしながら、式4.47で導かれた保存量は依然として奇妙です。開始時刻を適切に選ぶことで、この量はゼロにすることができます。これは保存されるので、あらゆる場合においてゼロのままです。
